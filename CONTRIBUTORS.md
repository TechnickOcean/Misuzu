## How to Get Listed Here

To be added to this list, you can:

1. **Contribute Code** - Submit pull requests with bug fixes, features, or improvements
2. **Improve Documentation** - Help write or improve guides, tutorials, and API docs
3. **Report Issues** - Find and report bugs with clear descriptions
4. **Answer Questions** - Help other community members in discussions and issues
5. **Provide Feedback** - Share your thoughts on features and design

### Contribution Types

We recognize all kinds of contributions:

- 💻 Code (features, fixes, refactoring)
- 📚 Documentation (guides, tutorials, API docs)
- 🐛 Bug reports (finding and reporting issues)
- 💬 Community (answering questions, helping others)
- 🎨 Design (UI/UX improvements)
- 🧪 Testing (writing and improving tests)
- 📢 Advocacy (promoting and discussing the project)

## Recognition

Contributors are recognized in multiple places:

- **This File** - Listed as project contributors
- **GitHub** - Automatically tracked as a repository contributor
- **Release Notes** - Mentioned in changelog for significant contributions
- **Discord/Community** - Recognized in community channels

## Guidelines

When contributing:

1. Follow our [Code of Conduct](./CODE_OF_CONDUCT.md)
2. Read [CONTRIBUTING.md](./CONTRIBUTING.md) for guidelines
3. Make sure your contributions align with project goals
4. Be respectful and constructive in all interactions

If you have questions about contributing or want to know how to get recognized, open a discussion in [GitHub Discussions](https://github.com/author/misuzu/discussions).
