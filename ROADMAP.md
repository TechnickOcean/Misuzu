## English

This document outlines the planned development roadmap for Misuzu.

### Version Strategy

Misuzu follows [Semantic Versioning](https://semver.org/):

- **MAJOR** (1.0.0): Breaking changes
- **MINOR** (0.1.0): New features
- **PATCH** (0.0.1): Bug fixes

### Current Status: v0.x (Pre-release)

Misuzu is currently in active development. The current version is **0.0.0** (alpha).

### Planned Releases

#### v0.1.0 - Core Stabilization

**Target**: Q2 2026

Features:

- Stabilize core API
- Improve documentation
- Add comprehensive test suite
- Performance optimizations

Deliverables:

- [ ] API stability guarantees
- [ ] 80%+ test coverage
- [ ] Performance benchmarks
- [ ] Migration guides

#### v0.2.0 - Enhanced Tooling

**Target**: Q3 2026

Features:

- CLI improvements
- Better error messages
- Plugin system enhancements
- Development tools

Deliverables:

- [ ] Improved CLI interface
- [ ] Plugin documentation
- [ ] Debug tools
- [ ] Visual debugging support

#### v1.0.0 - Production Ready

**Target**: Q4 2026

Features:

- Production-ready stability
- Long-term support commitment
- Extensive documentation
- Community ecosystem

Deliverables:

- [ ] Stable API with SemVer
- [ ] 3+ months stability testing
- [ ] Comprehensive guides
- [ ] Migration path defined

### Future Roadmap (Post-1.0)

#### 1.1.0 - Performance Focus

- Advanced caching strategies
- Build optimization
- Memory profiling tools
- Bundle analysis

#### 1.2.0 - Developer Experience

- Better error messages
- Development dashboard
- Debugging tools
- Community contributions

#### 2.0.0 - Major Enhancements

- Major architectural improvements
- Possibly breaking changes
- New paradigms/patterns
- Multi-language support (if applicable)
